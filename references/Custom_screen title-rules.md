## Scope

This rule applies only to the following components.

*  screen title
***

### ✅ Good Examples (Compliant with the Rules)

*   Adding a user
*   Setting permissions
*   Canceling an application
*   Output of reports
*   Editing data
*   Registering an account

👉 Points

*   All are expressed as **noun phrases**
*   Consistently follow the “object + of + noun” structure
*   Do not use verb forms (e.g., “do,” “perform”)

***

### ❌ Bad Examples (Contain Rule Violations)

*   Add a user
*   Set permissions
*   Cancel an application
*   Output a report
*   Edit data
*   Account registration

👉 Issues

*   Contain verbs (e.g., “Add,” “Output”)
*   Use inappropriate particles/structures (e.g., verb-object form)
*   Do not end as noun phrases
*   Missing “of,” making relationships unclear
